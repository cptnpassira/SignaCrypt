# ✍️ SignaCrypt

SignaCrypt is a CLI tool for generating ECDSA key pairs, signing files, and verifying digital signatures.

---

## 🚀 Features

- ECDSA (SECP256K1)
- SHA-256 hashing
- PEM key format
- File signature verification
- Simple CLI interface

---

## 📦 Installation

```bash
git clone https://github.com/cptnpassira/signacrypt.git
cd signacrypt
